# More pointers, arrays and strings

This project continues exploring advanced concepts in C programming:

## Learning Objectives
- Advanced pointer manipulation
- More complex string operations
- Array handling and manipulation
- Memory management concepts
- String parsing and processing

## Files
- `main.h` - Header file containing function prototypes
- Various `.c` files implementing advanced string and array functions

## Requirements
- All files compiled with gcc using flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Code follows Betty style guidelines
- No global variables allowed
- Maximum 5 functions per file
- Only `_putchar` allowed from standard library
